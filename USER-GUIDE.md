<a name="introduction"></a>
## 1️⃣ مقدمة / Introduction

### عن الموقع / About the Website

هذا الموقع عبارة عن منصة تفاعلية شاملة لرسالة ماجستير في العلاقات الدولية تدمج الواقعية البنيوية ونظريات المعلومات لتحليل 28 حالة تاريخية من 1910 إلى 2022.

**This website is a comprehensive interactive platform for a Master's thesis in International Relations that integrates Structural Realism and Information Theories to analyze 28 historical cases from 1910 to 2022.**

### الأهداف / Objectives

✅ عرض البحث الأكاديمي بطريقة تفاعلية  
✅ توفير أدوات تحليل بيانات متقدمة  
✅ تسهيل الوصول إلى المراجع الأكاديمية  
✅ دعم اللغتين العربية والإنجليزية  

✅ Present academic research interactively  
✅ Provide advanced data analysis tools  
✅ Facilitate access to academic references  
✅ Support Arabic and English languages

## 4️⃣ الميزات التفاعلية / Interactive Features

### الرسوم البيانية / Charts

#### كيفية الاستخدام / How to Use:

1. **التمرير فوق البيانات / Hover Over Data**
   - قم بتمرير المؤشر فوق أي نقطة بيانات لعرض القيمة الدقيقة

2. **النقر على الأسطورة / Click on Legend**
   - انقر على أي عنصر في الأسطورة لإخفاء/إظهار مجموعة البيانات

3. **التكبير/التصغير / Zoom In/Out**
   - استخدم عجلة الفأرة للتكبير والتصغير (في بعض الرسوم)

### الخرائط / Maps

#### كيفية الاستخدام / How to Use:

1. **التنقل / Navigation**
   - **السحب**: اسحب الخريطة للتنقل
   - **التكبير**: استخدم أزرار +/- أو عجلة الفأرة
   - **النقر**: انقر على العلامات لعرض التفاصيل

2. **خط زمني الحالات / Cases Timeline** (خريطة هيكل النظام)
   - **شريط التمرير**: حرك الشريط للتنقل عبر الحالات
   - **أزرار الأسهم**: استخدم ← → للانتقال بين الحالات
   - **التشغيل التلقائي**: انقر على زر التشغيل لعرض تلقائي

3. **التصفية / Filtering**
   - **الكل**: عرض جميع الحالات (28)
   - **حرب**: عرض حالات الحرب فقط (14)
   - **سلام**: عرض حالات السلام فقط (14)

### شبكة الاستشهادات / Citation Network

#### كيفية الاستخدام / How to Use:

1. **التفاعل مع العقد / Interact with Nodes**
   - **النقر**: انقر على أي عقدة لتمييز روابطها
   - **السحب**: اسحب العقد لإعادة ترتيب الشبكة
   - **التمرير**: مرر فوق العقدة لعرض معلوماتها

2. **البحث والتصفية / Search and Filter**
   - **البحث**: استخدم مربع البحث للعثور على مرجع محدد
   - **التصفية**: اختر مجال بحثي لعرض المراجع ذات الصلة

3. **التكبير/التصغير / Zoom**
   - استخدم عجلة الفأرة أو أزرار التكبير

## 5️⃣ تبديل اللغة / Language Switching

### كيفية التبديل بين اللغات / How to Switch Languages

#### من الصفحة الرئيسية / From Homepage:

**الطريقة 1: زر اللغة / Method 1: Language Button**
1. ابحث عن زر "العربية" في أعلى يمين الصفحة
2. انقر عليه للانتقال إلى النسخة العربية
3. Look for "العربية" button in top right
4. Click it to switch to Arabic version

**الطريقة 2: شريط التنقل / Method 2: Navigation Bar**
1. في شريط التنقل، ابحث عن أيقونة اللغة 🌐
2. انقر عليها لعرض خيارات اللغة
3. اختر اللغة المطلوبة

#### من أي صفحة أخرى / From Any Other Page:

- كل صفحة لها زر تبديل لغة مخصص
- ابحث عن "EN" أو "AR" في الزاوية العلوية
- Every page has a dedicated language toggle button
- Look for "EN" or "AR" in the top corner

### س7: ما معنى المتغيرات S, I, C؟ / Q7: What do S, I, C variables mean?

**S (Structural Constraint / القيد البنيوي)**
- قوة القيود الهيكلية في النظام الدولي
- Strength of structural constraints in the international system
- المدى: 0-10 / Range: 0-10

**I (Information Quality / جودة المعلومات)**
- جودة وتوفر المعلومات للدول
- Quality and availability of information to states
- المدى: 0-10 / Range: 0-10

**C (Convergence / التقارب)**
- درجة التقارب بين تصورات الدول
- Degree of convergence between states' perceptions
- المدى: 0-10 / Range: 0-10

**ج:** 28 حالة تاريخية من 1910 إلى 2022  
**A:** 28 historical cases from 1910 to 2022

- 14 حالة سلام / 14 peace cases

## 7️⃣ الدعم الفني / Technical Support

### المشاكل الشائعة وحلولها / Common Issues and Solutions

#### المشكلة 1: الموقع لا يُحمّل / Issue 1: Website Not Loading
**الحل / Solution:**
1. تحقق من اتصال الإنترنت / Check internet connection
2. امسح ذاكرة التخزين المؤقت / Clear browser cache
3. جرب متصفح آخر / Try another browser
4. أعد تحميل الصفحة (Ctrl+F5 / Cmd+R)

#### المشكلة 2: الرسوم البيانية لا تظهر / Issue 2: Charts Not Showing
**الحل / Solution:**
1. تأكد من تمكين JavaScript / Ensure JavaScript is enabled
2. عطّل مانع الإعلانات مؤقتاً / Disable ad blocker temporarily
3. حدّث المتصفح / Update browser
4. تحقق من إعدادات الأمان / Check security settings

#### المشكلة 3: الخرائط لا تعمل / Issue 3: Maps Not Working
**الحل / Solution:**
1. تحقق من سرعة الإنترنت / Check internet speed
2. انتظر تحميل المكتبات / Wait for libraries to load
3. أعد تحميل الصفحة / Reload the page
4. جرب جهاز آخر / Try another device

#### المشكلة 4: زر اللغة لا يعمل / Issue 4: Language Button Not Working
**الحل / Solution:**
1. تأكد من النقر على الزر الصحيح / Ensure clicking correct button
2. انتظر تحميل الصفحة كاملاً / Wait for page to fully load
3. استخدم الرابط المباشر للنسخة المطلوبة / Use direct link to desired version
