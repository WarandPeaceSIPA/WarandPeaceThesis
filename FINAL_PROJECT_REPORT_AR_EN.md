#### المحتوى الرياضي | Mathematical Content:

##### **القسم 1: النموذج البحثي الأساسي | Core Research Model**
1. **المعادلة 1**: الفرضية البحثية الرئيسية  
   `Outcome = f(S, I, C)` و `S + I → C → Peace/War`
   
2. **المعادلة 2**: دالة التقارب  
   `C = αS + βI + γ(S × I)`  
   **الفكرة الرئيسية**: الحد التفاعلي (S × I) يثبت أن جودة المعلومات (I) تضخم تأثير القيود البنيوية (S)

##### **القسم 2: إطار نظرية المعلومات | Information Theory Framework**
3. **المعادلة 3**: إنتروبيا شانون  
   `H(X) = -Σ p(xi) log₂ p(xi)`  
   **التطبيق**: قياس عدم اليقين - إنتروبيا أقل = جودة معلومات أعلى

4. **المعادلة 4**: تباعد كولباك-ليبلر  
   `D_KL(P || Q) = Σ P(i) log [P(i)/Q(i)]`  
   **التطبيق**: قياس الفرق في التوزيعات الاحتمالية - D_KL أقل = تقارب أعلى

5. **المعادلة 5**: المعلومات المتبادلة  
   `I(X;Y) = Σ p(x,y) log [p(x,y) / p(x)p(y)]`  
   **الاكتشاف الحاسم**: قنوات اتصال أفضل = جودة معلومات أعلى = احتمال سلام أعلى

##### **القسم 3: التعلم البايزي | Bayesian Learning**
6. **المعادلة 6**: نظرية بايز  
   `P(H|E) = [P(E|H) · P(H)] / P(E)`  
   **الفكرة**: جودة معلومات عالية = تحديثات معتقدات دقيقة = تقارب

7. **المعادلة 7**: معدل تقارب المعتقدات  
   `dC/dt = λI · (C_max - C_t)`  
   **الاكتشاف الرئيسي**: معدل التقارب يتناسب طردياً مع جودة المعلومات (I)

##### **القسم 4: نظرية الألعاب | Game Theory**
8. **المعادلة 8**: توازن ناش  
   `ui(si*, s-i*) ≥ ui(si, s-i*) ∀si ∈ Si`  
   **التطبيق**: I عالية = حسابات توازن دقيقة = سلام

9. **المعادلة 9**: المنفعة المتوقعة  
   `EUi = Σ pj · ui(ai, aj)`  
   **الربط الحاسم**: I تحدد دقة الاحتمالات (pj)

10. **المعادلة 10**: نطاق المساومة (نموذج فيرون)  
    `BR = [p - cA, p + cB]`  
    **الاختراق البحثي**: نطاق المساومة يوجد فقط مع I عالية

##### **القسم 5: النموذج الإحصائي التجريبي | Empirical Statistical Model**
11. **المعادلة 11**: الانحدار اللوجستي  
    `P(Peace) = 1 / (1 + e^-(β₀ + βS·S + βI·I + βC·C))`  
    **النتيجة التجريبية**: βI لديها أعلى معامل = I أقوى مؤشر على السلام

12. **المعادلة 12**: دالة العتبة الحرجة  
    ```
    Outcome = {Peace if I ≥ 7
              {War   if I < 7
    ```  
    **الاكتشاف الرئيسي**: I ≥ 7 ضرورية وكافية للسلام (دقة 100%)
