## 🔐 Security Notes

### Admin Panel
- **Default Password**: `admin123`
- **⚠️ IMPORTANT**: Change password immediately!
- **Access Method**: Ctrl+Shift+A keyboard shortcut
- **Login Page**: `admin-login.html`
- **Session**: 24-hour expiration

### Password Change Instructions
1. Open `admin-login.html` in text editor
2. Find line: `const ADMIN_PASSWORD_HASH = '...';`
3. Generate new SHA-256 hash of your password
4. Replace the hash value
5. Save and re-upload

**Generate SHA-256 Hash**: Use online tool or:
```javascript
// In browser console:
crypto.subtle.digest('SHA-256', 
  new TextEncoder().encode('your-new-password'))
  .then(h => console.log(Array.from(new Uint8Array(h))
    .map(b => b.toString(16).padStart(2,'0')).join('')))
```
